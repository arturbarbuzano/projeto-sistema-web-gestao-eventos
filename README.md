# Projeto Sistema Web Gestão de Eventos

Este projeto foi desenvolvido com o objetivo de gerenciar eventos e suas respectivas categorias de forma simples e organizada.

O sistema permite:

- Cadastrar categorias de eventos;
- Cadastrar eventos vinculados a uma categoria;
- Listar todos os eventos cadastrados;
- Visualizar informações como nome, categoria, data, local e descrição do evento.

## Tecnologias Utilizadas

- Java
- Spring Boot
- JPA
- Thymeleaf
- HTML5
- CSS3
- PostgreSQL

## Objetivo

O projeto foi desenvolvido para fins acadêmicos, aplicando conceitos de desenvolvimento web com Spring Boot, integração com banco de dados, arquitetura MVC e criação de interfaces responsivas para gerenciamento de informações.
